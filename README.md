# HomeLab

This repository documents the setup and management of my personal homelab environment. It includes
instructions, configuration files, supporting tools, and the applications deployed within the lab.

## Directory Structure

```
HomeLab/
├── documentation/       # Guides, setup instructions, and runbooks for the homelab
└── terraform/           # Infrastructure as Code (IaC) for provisioning homelab resources
```

### `documentation/`
Contains all written guides, setup instructions, and operational runbooks. This includes
step-by-step walkthroughs for configuring hardware, networking, services, and any other
components of the homelab.

### `terraform/`
Infrastructure as Code definitions used to provision and manage homelab resources (e.g. VMs,
networking, storage). Written using [Terraform](https://www.terraform.io/) /
[OpenTofu](https://opentofu.org/).

## Getting Started

Refer to the [`documentation/`](./documentation) directory for step-by-step instructions on
setting up and managing the homelab.
